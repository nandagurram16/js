1.DOM
A. Dom is Document Object Model(DOM). it is an APplication Programming interface(API) for manupulating Html Documnets.
   The DOm provides functions that allow you to add, remove, and modify parts of the document effectively.
   The dom represents an HTML Documnets as a tree of nodes.

3.Selecting elements in JS DOM.
A. -->getElementById()
    This method returns an element object that represents an HTML element.
   -->getElementsByName()
   This will returns the list of input names.

   This method accepts a tag name and returns a live HTMLCollection of elements.
   -->getElementsByClassName()
   This method returns an array--like of objects of the child elements with a specified class name.

   This method allows you to select the first element that matches one or more css selectors.
